Academic Term Tracking Mobile Application

This mobile application was developed to help students manage academic terms, courses, and assessments in a centralized system. The application allows users to organize coursework, track deadlines, and receive notifications for important academic events.

Features

  • Create, edit, and delete academic terms
  • Manage courses within each term
  • Track objective and performance assessments
  • Search courses using keyword filtering
  • Generate reports with structured academic data
  • Receive notifications for important dates
  • Add and share course notes

Technology Stack

  • .NET MAUI (Cross-platform mobile development)
  • C# Programming Language
  • SQLite Database
